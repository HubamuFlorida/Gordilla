Soy Hugo Balaguer, Desarrollador Web con visión de negocio. Tras graduarme en Administración y Dirección de Empresas (ADE), decidí combinar mis habilidades de gestión con mi pasión por la tecnología a través del Desarrollo de Aplicaciones Web. No solo creo sitios web funcionales, sino que entiendo los objetivos comerciales que hay detrás de ellos.

Conocimientos
JavaScript

Tengo experiencia sólida utilizando JavaScript para dotar de interactividad y dinamismo a las páginas web.

Soy capaz de manipular el DOM en tiempo real, gestionar eventos de usuario y desarrollar lógica compleja en el lado del cliente, lo que me permite crear experiencias de navegación fluidas y modernas sin necesidad de recargas constantes.

Conocimientos
PHP

En el desarrollo backend, utilizo PHP para gestionar toda la lógica del servidor y procesar datos de forma eficiente.

Tengo capacidad para conectar las aplicaciones con bases de datos y realizar operaciones CRUD, permitiendo crear sitios web dinámicos y funcionales donde la gestión de la información es segura, rápida y escalable según las necesidades del proyecto.

Conocimientos
CSS

Domino las hojas de estilo para transformar estructuras básicas en interfaces visuales atractivas y profesionales.

Me especializo en el diseño responsivo (Responsive Design), asegurando que cada web se adapte perfectamente a móviles y ordenadores. Utilizo sistemas modernos como Flexbox y Grid para maquetar contenidos de forma ordenada y estética.

Conocimientos
HTML

Considero HTML la columna vertebral del desarrollo web, por lo que priorizo siempre una estructura semántica, limpia y ordenada.

Me aseguro de cumplir con los estándares web para garantizar la máxima compatibilidad entre navegadores y la accesibilidad. Una buena arquitectura HTML es mi base fundamental para facilitar el posicionamiento SEO.
